# NicheSimulation.cpp

## Prerequisite(s)

* g++ compiler

## Instructions

To compile:

```
$ g++ NicheSimulation.cpp -o NicheSimulation
```

To run:

```
$ ./NicheSimulation
```

Output:

```
bigUrn_test.csv
```
